Studio

Pode escolher uma das seguintes opções para editar o SugarPortal:

  • Editar um módulo. Selecione esta opção para editar os módulos de Rastreador de Bugs, Ocorrências ou Potenciais
  • Carregar uma Style Sheet. Selecione esta opção para carregar uma stylesheet no SugarPortal.
  • Sincronizar o Portal. Selecione esta opção para sincronizar o Portal com o Sugar.

Para editar um módulo, faça o seguinte:

  1. Selecione o módulo que deseja reconfigurar no SugarPortal.
  2. Selecione a vista que deseja alterar.
    O Editor do Portal vai exibir uma barra de ferramentas no lado esquerdo e os campos de detalhe à direita. Para alterar a vista de Edição ou Listagem, selecione o link apropriado na parte superior do Painel. A barra de ferramentas vai exibir os campos disponíveis que atualmente não estão na vista de detalhe. Também exibe um campo vazio e um campo em branco. Utilize o campo em branco para criar espaços na vista. Por exemplo, use o campo em branco ao lado da Descrição para reservar mais espaço para uma descrição detalhada. Use um campo vazio para reservar espaço no formulário para o futuro.
  3. Para acrescentar um campo na barra de ferramentas na vista de detalhe, clique em Adicionar Linha. Selecione o campo da barra de ferramentas e arraste-o para o campo vazio.
  4. Para remover um campo da vista, selecione-o, e arraste-o para a barra de ferramentas.
  5. Clique em Salvar para salvar as suas alterações.

Para carregar um Style Sheet, faça o seguinte:

  1. Selecione a aba para os módulos de Ocorrências, Rastreador de Bug, ou Newsletters.
  2. Clique em Procurar, localize a Style Sheet, e selecione-a.
  3. Clique em Upload para carregar o Style Sheet.

Para sincronizar o Portal com o servidor de Sugar, faça o seguinte:

  1. Insira o URL da instância do portal que quer atualizar e clique em Avançar.
  2. Insira o seu nome de usuario e senha e clique em Iniciar Sincronização.
